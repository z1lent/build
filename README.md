## Windows Sandbox

https://www.thewindowsclub.com/windows-sandbox/

**Hypervisor conflict with VMware Virtual Machines**

***Use VMWare***

On Elevated Command Prompt: `bcdedit /set hypervisorlaunchtype off` and restart 

***Use Windows Sandbox***

On Elevated Command Prompt: `bcdedit /set hypervisorlaunchtype auto` and restart
